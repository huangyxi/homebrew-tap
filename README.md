# Huangyxi Tap

## How do I install these formulae?

`brew install huangyxi/tap/<formula>`

Or `brew tap huangyxi/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "huangyxi/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
